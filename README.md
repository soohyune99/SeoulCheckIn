# SeoulCheckIn
JSP 프로젝트 [서울체크인]

## 프로젝트 목적

  - 서울에 처음 방문한 외국인, 서울에 살아갈 외국인들을 위한 거주, 일자리, 교육 추천 및 커뮤니티 플랫폼

## 프로젝트에서 맡은 역활 

  - 외국인들을 위한 수강 프로그램 조회 및 신청 서비스 구현
  - 외국인들이 서로 소통할 수 있는 커뮤니티 구현
  
## 구현 기술 스택
  - 개발환경
    - MyBatis (3.5.10)
    - MySQL (8.0.27)
    - JDK 11.0.12
  - 형상관리 서버
    - Tomcat v9
    - Git
    - AWS
    - ubuntu
  - 사용 언어
    - HTML, CSS, JS
    - jQuery
    - Java 11
    - JSP
  
  - JSTL, AJAX, Kakao 지도 api, Cool SMS api
